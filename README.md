# Roll the Dice - Python OOP Mini Project

A simple command-line **Roll the Dice game** buid using **Python** and **Object-Oriented Programming (OOP)** concepts. This project is designed as a beginner-friendly mini project and as a foundation for clean code structure in python.

---

## Features

- Roll two dice randomly
- Interactive command line gameplay
- Uses Object-Oriented Programming (OOP)
- Clean and readable code structure
- Easy to extend with new features

## How the game works

- The program asks the user whether they want to roll the dice.
- If the user enters `y`, two dice are rolled and displayed.
- If the user enters `n`, the game ends.
- Any invalid input prompts the user to enter a valid choice
